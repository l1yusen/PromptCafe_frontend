# 一、 数据库表概览

| 序号 | 表名                | 中文名称          | 说明                                                         |
| :--- | :------------------ | :---------------- | :----------------------------------------------------------- |
| 1    | `users`             | 用户账号表        | 存储用户基础账号、角色及登录状态。                           |
| 2    | `prompts`           | Prompt 主表       | 存储用户创建的提示词，不包含历史版本。                       |
| 3    | `prompt_versions`   | Prompt 版本记录表 | 存储提示词的历史快照，用于版本回溯与对比。                   |
| 4    | `ai_api_keys`       | 用户 AI 配置表    | 存储用户个人配置的 API Key（加密存储）。                     |
| 5    | `system_ai_configs` | 系统 AI 配置表    | 存储系统默认的 AI 接口配置（供游客或未配置 Key 的用户使用）。 |
| 6    | `ai_call_records`   | AI 调用日志表     | 记录所有 AI 润色、测试的调用详情及消耗。                     |
| 7    | `guest_ai_quotas`   | 游客调用额度表    | 专门用于控制和统计游客每日的调用次数。                       |
| 8    | `community_prompts` | 社区分享表        | 存储用户分享到社区并经过审核的提示词快照。                   |
| 9    | `user_favorites`    | 用户收藏表        | 记录用户收藏的社区 Prompt。                                  |
| 10   | `community_reports` | 社区举报表        | 存储用户对违规内容的举报信息及处理结果。                     |
| 11   | `audit_logs`        | 系统审计日志表    | 记录管理员的关键操作，确保系统行为可追溯。                   |

---

### 二、 详细字段设计

#### 1. 用户账号表 (users)
| 字段名          | 类型         | 约束                    | 说明                                 |
| :-------------- | :----------- | :---------------------- | :----------------------------------- |
| `id`            | UUID         | PK, 默认生成            | 用户唯一 ID                          |
| `username`      | VARCHAR(50)  | NOT NULL, UNIQUE        | 用户名                               |
| `email`         | VARCHAR(255) | NOT NULL, UNIQUE        | 邮箱                                 |
| `password_hash` | VARCHAR(255) | NOT NULL                | 加盐哈希后的密码                     |
| `role`          | VARCHAR(20)  | DEFAULT 'user'          | 角色：user(普通用户) / admin(管理员) |
| `status`        | VARCHAR(20)  | DEFAULT 'active'        | 状态：active(活跃) / disabled(禁用)  |
| `nickname`      | VARCHAR(50)  | NULL                    | 用户昵称                             |
| `avatar_url`    | TEXT         | NULL                    | 头像地址                             |
| `bio`           | TEXT         | NULL                    | 个人简介                             |
| `last_login_at` | TIMESTAMP    | NULL                    | 最近登录时间                         |
| `created_at`    | TIMESTAMP    | NOT NULL, DEFAULT NOW() | 注册时间                             |
| `updated_at`    | TIMESTAMP    | NOT NULL, DEFAULT NOW() | 资料更新时间                         |

#### 2. Prompt 主表 (prompts)
| 字段名            | 类型         | 约束                    | 说明                             |
| :---------------- | :----------- | :---------------------- | :------------------------------- |
| `id`              | UUID         | PK, 默认生成            | Prompt ID                        |
| `user_id`         | UUID         | FK(users.id), NOT NULL  | 创建者 ID                        |
| `title`           | VARCHAR(200) | NOT NULL                | 提示词标题                       |
| `description`     | TEXT         | NULL                    | 提示词简要描述                   |
| `system_prompt`   | TEXT         | NULL                    | 系统提示词                       |
| `user_prompt`     | TEXT         | NOT NULL                | 用户提示词正文                   |
| `variables`       | JSONB        | NULL, DEFAULT '[]'      | 变量定义（名称、标签、类型等）   |
| `tags`            | JSONB        | NULL, DEFAULT '[]'      | 标签列表（字符串数组）           |
| `visibility`      | VARCHAR(20)  | DEFAULT 'private'       | private(私有) / public(社区可见) |
| `current_version` | INTEGER      | DEFAULT 1               | 当前版本号                       |
| `created_at`      | TIMESTAMP    | NOT NULL, DEFAULT NOW() | 创建时间                         |
| `updated_at`      | TIMESTAMP    | NOT NULL, DEFAULT NOW() | 最后一次修改时间                 |

#### 3. Prompt 版本记录表 (prompt_versions)
| 字段名           | 类型         | 约束                              | 说明                           |
| :--------------- | :----------- | :-------------------------------- | :----------------------------- |
| `id`             | UUID         | PK, 默认生成                      | 版本快照 ID                    |
| `prompt_id`      | UUID         | FK(prompts.id), ON DELETE CASCADE | 关联的 Prompt ID               |
| `version_number` | INTEGER      | NOT NULL                          | 版本号                         |
| `title`          | VARCHAR(200) | NOT NULL                          | 该版本的标题                   |
| `description`    | TEXT         | NULL                              | 该版本的简要描述快照           |
| `system_prompt`  | TEXT         | NULL                              | 该版本的系统提示词             |
| `user_prompt`    | TEXT         | NOT NULL                          | 该版本的用户提示词正文         |
| `variables`      | JSONB        | NULL                              | 该版本的变量快照               |
| `tags_snapshot`  | JSONB        | NULL, DEFAULT '[]'                | 该版本的标签快照（字符串数组） |
| `note`           | TEXT         | NULL                              | 版本变更备注                   |
| `created_at`     | TIMESTAMP    | NOT NULL, DEFAULT NOW()           | 版本创建时间                   |

#### 4. 用户 AI 配置表 (ai_api_keys)
| 字段名              | 类型         | 约束                    | 说明                           |
| :------------------ | :----------- | :---------------------- | :----------------------------- |
| `id`                | UUID         | PK, 默认生成            | 配置 ID                        |
| `user_id`           | UUID         | FK(users.id), UNIQUE    | 所属用户                       |
| `provider`          | VARCHAR(50)  | NOT NULL                | AI 服务商 (openai, deepseek等) |
| `base_url`          | TEXT         | NOT NULL                | API 代理/基础地址              |
| `encrypted_api_key` | TEXT         | NOT NULL                | 加密后的 API Key               |
| `key_mask`          | VARCHAR(50)  | NOT NULL                | 脱敏后的 Key (如 sk-***abcd)   |
| `default_model`     | VARCHAR(100) | NOT NULL                | 默认使用的模型                 |
| `is_verified`       | BOOLEAN      | NOT NULL, DEFAULT FALSE | 最近一次连通性校验是否成功     |
| `last_verified_at`  | TIMESTAMP    | NULL                    | 最近一次校验时间               |
| `status`            | VARCHAR(20)  | DEFAULT 'active'        | active / disabled              |
| `created_at`        | TIMESTAMP    | NOT NULL, DEFAULT NOW() | 创建时间                       |
| `updated_at`        | TIMESTAMP    | NOT NULL, DEFAULT NOW() | 更新时间                       |

#### 5. 系统 AI 配置表 (system_ai_configs)
| 字段名              | 类型         | 约束                    | 说明                            |
| :------------------ | :----------- | :---------------------- | :------------------------------ |
| `id`                | UUID         | PK, 默认生成            | 系统配置 ID                     |
| `provider`          | VARCHAR(50)  | NOT NULL                | 服务商名称                      |
| `base_url`          | TEXT         | NOT NULL                | 系统默认 API 地址               |
| `key_mask`          | VARCHAR(50)  | NOT NULL                | 脱敏后的系统 Key (用于后台展示) |
| `encrypted_api_key` | TEXT         | NOT NULL                | 加密存储的系统 Key              |
| `default_model`     | VARCHAR(100) | NOT NULL                | 系统默认模型                    |
| `is_default`        | BOOLEAN      | DEFAULT FALSE           | 是否为当前生效的默认配置        |
| `is_enabled`        | BOOLEAN      | DEFAULT TRUE            | 是否启用                        |
| `daily_guest_limit` | INTEGER      | DEFAULT 10              | 游客单日调用次数上限            |
| `created_by`        | UUID         | FK(users.id)            | 创建该配置的管理员 ID           |
| `updated_by`        | UUID         | FK(users.id)            | 最近修改该配置的管理员 ID       |
| `created_at`        | TIMESTAMP    | NOT NULL, DEFAULT NOW() | 记录创建时间                    |
| `updated_at`        | TIMESTAMP    | NOT NULL, DEFAULT NOW() | 更新时间                        |

#### 6. AI 调用日志表 (ai_call_records)
| 字段名 | 类型 | 约束 | 说明 |
| :--- | :--- | :--- | :--- |
| id | UUID | PK, 默认生成 | 调用记录 ID |
| user_id | UUID | FK(users.id), NULL | 调用用户（游客为 NULL） |
| guest_session_id | VARCHAR(100) | NULL | 游客会话 ID |
| prompt_id | UUID | FK(prompts.id), NULL | 关联的个人 Prompt |
| community_prompt_id| UUID | FK(community_prompts.id) | 关联的社区 Prompt |
| call_type | VARCHAR(20) | NOT NULL | polish(润色) / test(测试) |
| api_key_source | VARCHAR(20) | NOT NULL | user(用户私有) / system_default(系统) |
| provider | VARCHAR(50) | NOT NULL | AI 服务商 |
| model | VARCHAR(100) | NOT NULL | 调用的模型名称 |
| input_summary | TEXT | NULL | 输入摘要，用于列表展示 |
| rendered_system_prompt| TEXT | NULL | 渲染后的系统提示词（包含变量替换后） |
| rendered_user_prompt| TEXT | NULL | 渲染后的用户提示词（包含变量替换后） |
| output_text | TEXT | NULL | AI 输出的完整内容 |
| suggestions | JSONB | NULL | 润色建议列表（仅 polish 类型有效） |
| variables | JSONB | NULL | 测试时使用的变量值快照 |
| temperature | DECIMAL(3,2)| NULL | 模型温度参数 |
| max_tokens | INTEGER | NULL | 设定的最大 Token 数 |
| prompt_tokens | INTEGER | NULL | 输入消耗 Token 数 |
| completion_tokens | INTEGER | NULL | 输出生成 Token 数 |
| total_tokens | INTEGER | NULL | 总 Token 消耗 |
| latency_ms | INTEGER | NULL | 响应耗时（毫秒） |
| status | VARCHAR(20) | DEFAULT 'success' | success / failed |
| error_code | VARCHAR(50) | NULL | 错误码 |
| error_message | TEXT | NULL | 错误详情 |
| ip_address | VARCHAR(100)| NULL | 调用者 IP |
| user_agent | TEXT | NULL | 浏览器或客户端信息 |
| created_at | TIMESTAMP | NOT NULL | 调用时间 |

#### 7. 游客调用额度表 (guest_ai_quotas)
| 字段名 | 类型 | 约束 | 说明 |
| :--- | :--- | :--- | :--- |
| id | UUID | PK, 默认生成 | 记录 ID |
| guest_session_id | VARCHAR(100) | NOT NULL | 游客会话 ID |
| ip_address | VARCHAR(100) | NULL | 游客 IP |
| usage_date | DATE | NOT NULL | 统计日期 |
| polish_count | INTEGER | DEFAULT 0 | 当日 AI 润色调用次数 |
| test_count | INTEGER | DEFAULT 0 | 当日 AI 测试调用次数 |
| total_count | INTEGER | DEFAULT 0 | 当日总调用次数 |
| daily_limit | INTEGER | DEFAULT 10 | 每日上限 |
| last_called_at | TIMESTAMP | NULL | 最近调用时间 |
| created_at | TIMESTAMP | NOT NULL, DEFAULT NOW() | 创建时间 |
| updated_at | TIMESTAMP | NOT NULL, DEFAULT NOW() | 更新时间 |

#### 8. 社区分享表 (community_prompts)
| 字段名                   | 类型         | 约束                    | 说明                                                       |
| :----------------------- | :----------- | :---------------------- | :--------------------------------------------------------- |
| `id`                     | UUID         | PK, 默认生成            | 分享 ID                                                    |
| `prompt_id`              | UUID         | FK(prompts.id)          | 原始提示词 ID（用于追踪来源）                              |
| `user_id`                | UUID         | FK(users.id)            | 分享者 ID                                                  |
| `title_snapshot`         | VARCHAR(200) | NOT NULL                | 分享时的标题快照                                           |
| `system_prompt_snapshot` | TEXT         | NULL                    | 分享时的系统提示词快照                                     |
| `user_prompt_snapshot`   | TEXT         | NOT NULL                | 分享时的用户提示词快照                                     |
| `tags_snapshot`          | JSONB        | NULL                    | 分享时的标签快照                                           |
| `description`            | TEXT         | NULL                    | 用户填写的分享描述信息                                     |
| `status`                 | VARCHAR(20)  | DEFAULT 'pending'       | pending(待审)/approved(通过)/rejected(驳回)/removed (下架) |
| `reviewed_by`            | UUID         | FK(users.id)            | 审核人                                                     |
| `reviewed_at`            | TIMESTAMP    | NULL                    | 审核时间                                                   |
| `view_count`             | INTEGER      | DEFAULT 0               | 浏览次数                                                   |
| `favorite_count`         | INTEGER      | DEFAULT 0               | 收藏总数                                                   |
| `audit_note`             | TEXT         | NULL                    | 审核意见                                                   |
| `created_at`             | TIMESTAMP    | NOT NULL, DEFAULT NOW() | 分享提交时间                                               |

#### 9. 用户收藏表 (user_favorites)
| 字段名                | 类型      | 约束                                        | 说明              |
| :-------------------- | :-------- | :------------------------------------------ | :---------------- |
| `id`                  | UUID      | PK                                          | 收藏记录 ID       |
| `user_id`             | UUID      | FK(users.id), ON DELETE CASCADE             | 收藏者用户 ID     |
| `community_prompt_id` | UUID      | FK(community_prompts.id), ON DELETE CASCADE | 收藏的社区作品 ID |
| `created_at`          | TIMESTAMP | NOT NULL, DEFAULT NOW()                     | 收藏时间          |

#### 10. 社区举报表 (community_reports)
| 字段名                | 类型         | 约束                     | 说明                                         |
| :-------------------- | :----------- | :----------------------- | :------------------------------------------- |
| `id`                  | UUID         | PK, 默认生成             | 举报 ID                                      |
| `reporter_id`         | UUID         | NULLABLE                 | 举报人（匿名游客可为 NULL）                  |
| `community_prompt_id` | UUID         | FK(community_prompts.id) | 被举报的社区作品                             |
| `reason`              | VARCHAR(200) | NOT NULL                 | 举报原因（选项）                             |
| `description`         | TEXT         | NULL                     | 补充说明                                     |
| `status`              | VARCHAR(20)  | DEFAULT 'pending'        | pending / processed(已处理) / rejected(驳回) |
| `handled_by`          | UUID         | FK(users.id)             | 处理的管理员                                 |
| `handled_at`          | TIMESTAMP    | NULL                     | 处理时间                                     |
| `handle_result`       | TEXT         | NULL                     | 处理结果说明                                 |
| `created_at`          | TIMESTAMP    | NOT NULL, DEFAULT NOW()  | 举报时间                                     |

#### 11. 系统审计日志表 (audit_logs)
| 字段名        | 类型         | 约束                    | 说明                                 |
| :------------ | :----------- | :---------------------- | :----------------------------------- |
| `id`          | UUID         | PK, 默认生成            | 日志 ID                              |
| `actor_id`    | UUID         | FK(users.id)            | 操作者（管理员）ID                   |
| `actor_role`  | VARCHAR(20)  | NULL                    | 操作人角色                           |
| `action`      | VARCHAR(100) | NOT NULL                | 动作 (audit_approve, user_disable等) |
| `target_type` | VARCHAR(50)  | NOT NULL                | 目标类型 (user, prompt, report)      |
| `target_id`   | UUID         | NOT NULL                | 目标 ID                              |
| `detail`      | JSONB        | NULL                    | 详细信息（修改前后的值快照）         |
| `ip_address`  | VARCHAR(100) | NULL                    | 操作者 IP 地址                       |
| `user_agent`  | TEXT         | NULL                    | 浏览器或客户端信息                   |
| `created_at`  | TIMESTAMP    | NOT NULL, DEFAULT NOW() | 记录时间                             |
